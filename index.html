📤 Import / Export

💾 Backup dati

Esporta tutti i dati in un file JSON. Conserva questo file per riaprire l'app su un altro PC o dopo aver svuotato la cache del browser.

↩ Ripristina da backup

Carica un file JSON di backup per ripristinare tutti i dati. Lo stato attuale verrà salvato nella cronologia undo.

📋 Aggiorna da Excel

Importa o aggiorna i dati da un file Formazione_obbligatoria.xlsx. I dati esistenti vengono uniti (merge) con quelli importati.

🗑 Cancella tutti i dati

Rimuove tutti i dati salvati nel browser. Questa operazione non può essere annullata — assicurati di avere un backup JSON.
